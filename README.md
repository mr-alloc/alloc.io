## 테이블 스타일 옵션

### 표 정렬

```
:{ "wrapper-class": "items-center" } //옵션 적용예시 "wrapper-class"로 적용
```
|  방향  | Description  |
|--------|--------------|
|  Left  | items-start  |
| Center | items-center |
| Right  | items-end    |

### 표 설명 추가

```
:{ "description": "표 설명" }
```

## 이미지 스타일 옵션

### 이미지 정렬

```
![이미지](이미지경로)
:{ "wrapper-class": "justify-center" } //옵션 적용예시 "wrapper-class"로 적용
```
|  방향  | Description  |
|--------|--------------|
|  Left  | justify-start  |
| Center | justify-center |
| Right  | justify-end    |
